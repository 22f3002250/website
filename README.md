# Tripti Kumari - Portfolio Website

A modern, animated portfolio website showcasing the journey of a Full Stack Developer & AI/ML Engineer.

## Theme: "Code. Create. Explore."

A premium storytelling experience combining technical expertise with creative personality.

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion (animations)
- Vite (build tool)
- Lucide React (icons)

## Features

- Smooth scroll animations
- Glassmorphism design
- Gradient text effects
- Interactive hover states
- Responsive design
- Dark theme with electric blue, teal, and purple accents

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Sections

- Hero: Bold statement with typing animation
- Journey: Storytelling about section
- Technical Map: Skills organized as territories
- Projects: Cinematic project showcases
- Explorer: Travel personality
- Creative Side: Hobbies and creativity
- Timeline: Educational milestones
- Contact: Connection section

## Customization

Update personal information in the component files:
- Contact details in `src/components/Contact.jsx`
- Projects in `src/components/Projects.jsx`
- Timeline events in `src/components/Timeline.jsx`
